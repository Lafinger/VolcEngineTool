# 第二步 封装好接口后，帮把 音频文本案例.md 里的文本进行语音合成，合成规则如下：
- 合成音频文件格式为 wav
- 合成音频输出的目录与 tts_https.py 同级，为 tts_https_wavs
-  “#” 后面的文本不进行合成， 如： “# 主页面”， “# 存款取款转账页面”
- 对于“#”标题下的文本进行合成，合成文件名为 “#” 后面的文本，如：

您好，我是您的智能助手，欢迎来到深圳储蓄银行，请问您需要咨询什么业务？”， 合成文本为：“您好，我是您的智能助手，欢迎来到深圳储蓄银行，请问您需要咨询什么业务？”，合成的文件名为：主页面.wav
